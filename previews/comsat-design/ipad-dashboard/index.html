1. Background Labels

Render section names, slot labels, legend, and global command labels into the iPad background so each button can stay dense.

2. Smart Tiles

Keep one status tile per workspace. Tap approves if there is a pending permission; otherwise it focuses the workspace.

3. Dynamic Slots

Slots map to COMSAT snapshot order, sorted by urgency, so current windows appear and stale windows disappear without manual rewiring.

4. Optional Secondaries

Add visible focus/reject/top-approve buttons only in the command area, not as a mandatory duplicate for every window.

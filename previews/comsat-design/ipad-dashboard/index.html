1. Rows Are Stations

Use one row per live COMSAT window. Page A covers rows 1-8; Page B repeats the same layout for rows 9-16 when the live count spikes.

2. Column Grammar

Identity uses short operator codes with the full station name below. State is symbolic, detail is small/top-left, and actions are raised button caps.

3. Two Color Channels

Station color runs through the whole row as top/bottom rails and the identity plate. State colors only appear as glyphs, lamps, and state words.

4. Background Does The Grouping

The iPad background carries the column headers and row rails. The keys stay square, but the faceplate makes each row read as one control bay.

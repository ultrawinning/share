// design system / portfolio

A small, rigorous system. Sharp rectangles, one pink accent, three fonts.

Terminal-adjacent, dark-mode-first, anti-slop. Outputs the website (ultrawinning.com), IG stories, IG posts, carousels, and pull-quote graphics. Sourced from a claude.ai/design handoff bundle, dogfooded here.

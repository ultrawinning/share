One client, three stages — not one wall of everything

The coaching record holds a lot per client, and dumping it all in one panel is what makes it hard to use. This splits every client into the stage they're actually in — Onboarding, Check-in, Management — so a coach only sees what that moment needs. Same three stages drive the drawer tabs and the columns on each table. Nothing is rebuilt; the existing cards are re-grouped.

01

The three stages

Onboarding

Get them ready for call 1

Everything to run the first call on one screen — no hunting across tabs.

Holds sales-call notes · consultation · what they paid · targets
Default when started < 30d ago & not yet onboarded
List view Onboarding queue on /coaching
Check-in

The every-few-days loop

The steady coaching rhythm — who's due a touch, and the numbers to run it.

Holds touch-due timer · weights · workouts · short sales summary
Default when active & past onboarding (most clients)
List view /coach workstation & roster
Management

The retention stack

Keeping and closing the loop — the periodic + exit layer.

Holds start/end · retention · NPS · testimonial · re-sign
Default when day-60 window or an open ask
List view /coaching/retention

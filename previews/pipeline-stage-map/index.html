Pipeline Stage Map DIAGRAM DRAFT

Every working sub-stage on the three sales boards, and the funnel milestone it counts as. The 5 milestones are the only thing HPFHQ counts; each board has its own sub-stages, and every sub-stage rolls up to exactly one milestone. Duplicate labels (e.g. "Deposit Closed" / "Won" / "Closed") are now consolidated to one canonical sub-stage per board. A BRANCH card is an off-ramp (cancelled / no-show / lost) — it records what happened, it never moves a lead backwards.

How "adding a stage" works. Stages are defined in code (we keep control — no self-serve editor). To add one, add it to the map declaring which of the five milestones it sits under. A guard checks every live stage is mapped, so a stage can never exist without a milestone. This diagram is generated from that map.

UI Direction · redline before build

Refined, not decorated.

The base reads near-monochrome — grey hierarchy, tight type, hairline dividers do the structural work. The three stages (Onboarding · Check-in · Management) are the only color, and they appear only as thin accents: a 3px edge, a tab underline, a small dot, a pale tint on a badge. Never filled blocks. This is the anti-“clown-car” rule made concrete.

Monochrome base

Grey hierarchy (4 levels) + weight + spacing carry structure. Color is subtracted, not added.

Color = signal

A hue means one thing: which stage. It's a wayfinding accent, never wallpaper.

One interactive tone

Muted forest for links/active; brand-green stays logo-only. No rainbow of buttons.

Type does the work

Space Grotesk display · Inter body · tabular mono numerals. Hierarchy replaces boxes.

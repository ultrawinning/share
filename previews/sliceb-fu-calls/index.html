Violet 4px left-border + “↩ FOLLOW-UP” pill in every card state = the closer instantly sees it’s a follow-up, not a new booked call. Normal calls keep their teal (set) / grey (unclaimed) border.

The header counts only new booked calls (“1 call”); follow-ups are tallied separately (“2 follow-ups”) and never feed Calls-Scheduled / show-rate (R1 funnel-exclude).

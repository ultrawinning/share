Drawer rules

  • Top line = workspace color · double-thickness on Nexus cards
  • Status dot top-left: orange = cooking, yellow = VS Code needs attention, green = standing by
  • Shorthand = user-defined per workspace (HQc, CC, DBG, etc). Default if unset = first 2 letters of category
  • Click tile → focuses bound VS Code window
  • Hover sustained ~200ms → drawer expands to full work view
  • Drag the drawer itself → repositions along the edge, snaps to nearest edge

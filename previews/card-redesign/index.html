Design Notes

Qual indicator: Closer Today/Pipeline uses three-bar (all 3 signals relevant post-call). Setter Inbox/Pipeline uses single dot (only pre-call data exists). Won cards always show three-bar (full journey visible).

Setter badge: Shows setter name (replaces SET CALL + 👤). Tooltip: full name + booking date + channel. No setter → UNCLAIMED.

Platform icons: Left under name. IG (gradient), LI (blue), Meta AD (orange), MC (purple), Email (grey), Phone (teal), YT (red). Clickable where applicable.

NLS number: In qual summary line: "85 · CEO · $150K+ · Urgent". Green >70, amber 50-70, red <50, — if no data.

Objection: Single 🗣️ emoji for all objection types. Tooltip shows which: Price, Timing, Spouse, Think About It, Not Ready, Other.

Activity badge: Only text badge. Last contact method + days ago. Green ≤3d, yellow 4-7d, orange 8-14d, red 15d+.

SendBlue: Pending API integration research. Once connected, can auto-detect video/text confirmations by phone number.

Color System (v3.1 — no overlaps)

ColorMeaningWhere
GreenConfirmed / WonCloser Today, Setter Booked, Outcomes
YellowPending ReplyCloser Today (sent, awaiting)
OrangeNot ContactedCloser Today (nobody reached out yet)
BlueFresh / NewSetter Inbox (speed to lead)
TealActive / On TrackPipeline (on schedule)

RedOverdue / UrgentPipeline (past FU date, stale >7d)
GreyNeutral / DoneDefault resting state, submitted

Where to Pull Monthly Impressions

Dan wants total monthly impressions across all social platforms. Here's exactly where to find each one.

Quick summary:
✅ Pull right now (5): Instagram, Facebook, Twitter/X (HypeFury), LinkedIn (Taplio), TikTok
⚠️ Needs access from Dan (2): YouTube, Substack
❓ May not be available yet (1): Threads

Instagram You have access

1,501,138 followers (+10,810 this week)
Where to find impressions
Instagram app → Professional Dashboard → Accounts Reached
Steps
  1. Open Instagram → tap the profile picture
  2. Tap Professional dashboard (or the bar chart icon)
  3. Tap Accounts reached → set to Last 30 days
  4. The number at the top = total impressions for the month
  5. Screenshot or write down both Reach and Impressions if both are shown
Alternative
Meta Business Suite → Insights → Instagram → set date range → Impressions

Facebook You have access

536,901 followers (+14,229 this week)
Where to find impressions
Meta Business Suite → Insights → Overview
Steps
  1. Go to business.facebook.com
  2. Select the CoachDanGo page
  3. Click Insights in the left sidebar
  4. Click Overview → set date range to the current month
  5. Use the "Impressions" number (total views, not unique reach)

Twitter / X Use HypeFury

942,503 followers (+14,023 this week)
Where to find impressions
HypeFury (you already use this to post) → Analytics → Monthly impressions
Steps
  1. Log into HypeFury
  2. Go to Analytics or Dashboard
  3. Set date range to the current month
  4. Total impressions should be visible — HypeFury pulls this from the X API automatically
Alternative
x.com → More → Creator Studio → Analytics → set to current month

YouTube Studio → Analytics → Overview
Steps
  1. Go to studio.youtube.com
  2. Click Analytics in the left sidebar
  3. Set date range to the current month
  4. Use "Views" — this is YouTube's standard equivalent of impressions

TikTok You have access

32,160 followers (+1,631 this week)
Where to find impressions
TikTok app → Creator tools → Analytics → Overview
Steps
  1. Open TikTok → tap Profile
  2. Tap the ☰ menu → Creator tools → Analytics
  3. Tap Overview
  4. Use "Video views" — TikTok's equivalent of impressions
  5. Set to Last 28 days (closest to monthly)

Threads Limited analytics

~341,000 followers (+~1,000 this week)
Where to find impressions
Threads app → Profile → Insights
Steps
  1. Open Threads → tap Profile
  2. Look for Insights or View insights below the bio
  3. If available, it shows views/reach for the past 7 or 28 days
Fallback
Threads analytics is still limited. If monthly totals aren't shown, check Meta Business Suite (may aggregate Threads now). If neither works, note "not yet available" in the report.

Personal Project Tracker — build prompt

A lightweight, self-owned project board with deadlines. You build it in Claude Code, then manage your projects by just talking to it.

How to use

  1. Open Claude Code in a new empty folder.
  2. Hit Copy prompt below and paste the whole thing in.
  3. When it asks, say start with step 1 and run each step as it ships.
  4. Later, just tell it things like add a project: launch newsletter, high priority, due mid-August — it writes the files for you.

Friday 22 May — WD Elements migration playbook

Full procedural playbook with verification gates lives at ~/.claude/plans/friday-2026-05-22-wd-elements-migration.md on GHOST. Calendar event triggers at 07:00 WEST. Below is the high-level summary.

Pre-flight (when boxes arrive)

  • Unbox + plug both WDs into Anker Prime TB5 dock
  • SMART check both — diskutil info /dev/disk<N> | grep SMART
  • Both should report "SMART Status: Verified"

Sequenced steps

  1. Format both as APFS Encrypted via Disk Utility GUI. Names: Khaydarin-1 + Khaydarin-2. Store passphrases in 1Password.
  2. Migrate GHOST staging → Khaydarin-1 (rsync ~/Storage-staging/touro/ + the Pre-2015-Photos library). ~30-60 min.
  3. Re-plug touro + Orange LaCie + SanDisk. Direct-copy each → Khaydarin-1:
    • Orange LaCie 2.3 TB (~6-12 hr, background)
    • SanDisk's 1.5 TB FULL Iphoto library May 3rd 2026 (~3-6 hr)
  4. Seed Khaydarin-2 mirror from Khaydarin-1 (~6-12 hr, overnight)
  5. Install daily mirror launchd job firing 02:00 local. Plist label: com.ultrawinning.khaydarin-mirror.
  6. Verify both archives healthy: file counts, sample reads, size matches.
  7. NOW safe to wipe touro + Orange LaCie + PYLON. Disk Utility → APFS Encrypted (or factory exFAT if selling).
DO NOT WIPE touro / Orange / PYLON until Step 6 (mirror verification) passes. Single-copy data on Khaydarin-1 alone is not safe.

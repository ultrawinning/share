CONSOLE

Your HPF Sales Operations Assistant — powered by Claude

What is this?

CONSOLE is an AI assistant that knows HPF sales operations. It has curated knowledge about GHL workflows, sales metrics, the payment pipeline, Typeform qualification, and your scorecard. For HPF-specific questions, it answers from those files with source citations. For general platform questions (like "how do I do X in GHL"), it can also search the web and combine results with HPF context.

When it doesn't know something HPF-specific, it flags the gap for Emil. Over time, those gaps get filled and the system gets smarter. Think of it as your Claude that already knows how HPF works.

Getting started

1 Open Claude Code
Go to claude.ai/code and sign in with your Claude Team account.
2 Connect the repo
You should see HPF-Console-Sales in the repo selector at the bottom. If you don't, connect your GitHub account first — you'll only see this one repo.
3 Start a session
Open a new session and type /onboarding. The assistant will introduce itself and ask you a few questions about your role. Answer honestly — your answers help Emil improve the knowledge base. You only need to do this once.
4 Ask away
Ask any sales ops question. If the answer is in the knowledge files, you'll get it with a source citation. If not, it'll tell you and flag it.

How to use it well

Do
  • Ask specific questions
  • Ask about GHL workflows and config
  • Ask for help drafting your weekly report
  • Ask about metric formulas and targets
  • Ask "how does X work?" about any sales process
  • Ask general GHL/platform how-to questions
  • Use anything you'd use normal Claude for
Don't
  • Ask for API keys or webhook secrets
  • Ask it to write code or scripts
  • Paste passwords or sensitive credentials
  • Ask it to guess about HPF-specific config

Commands

/onboarding
First-time setup. Introduces the Console and asks about your role. Run once.
/wrap
End-of-session summary. Saves what you discussed, what was answered, and any gaps found. Use before closing a session.

When it can't answer

For general platform questions (like "how to connect WhatsApp to GHL"), it searches the web and gives you an answer with HPF context. For HPF-specific questions where the knowledge files don't have the answer, it flags the gap for Emil. The more you use it, the more gaps get identified and filled.

Escalation

If the assistant tells you to escalate to Emil, that means the question involves code, database, API integrations, webhook configuration, targets, compensation, or strategic decisions. Those are outside the ops knowledge base by design.
